# Exercícios de Manipulação de Arquivos em C

## Exercício 1
Escreva um programa em C para criar um arquivo de texto e escrever uma mensagem nele.

## Exercício 2

## Exercício 3

## Exercício 6

## Exercício 15

## Exercício 18

## Exercício 19

## Exercício 21
